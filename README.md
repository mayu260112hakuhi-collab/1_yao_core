# 八百万エンジン (Yaoyorozu Engine)

**日本語で動くrust**

## 概要
八百万エンジンは、Rustをベースとした日本語記述スクリプトエンジンです。
英語圏の言語に縛られず、日本語の思考プロセスをそのままコードとして実行可能な環境を提供します。

## 特徴
- **日本語ネイティブ**: 英語の翻訳作業を介さず、直感的にコードを記述。
- **Rustベース**: 高い安全性とパフォーマンスを維持。
- **シンプル設計**: 複雑な環境構築を最小化し、書くことに集中。
